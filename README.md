# React Webpack Esbuild starter

Enjoy blazing fast builds using Esbuild for your React app 🏎  
Minimal config ☕️ 

---

Key features:

- typescript
- react 17
- esbuild
- webpack + esbuild-loader

## Getting Started

```bash
git clone git@github.com:omachala/react-esbuild-starter.git myapp
cd myapp/
yarn upgrade-interactive
yarn install
rm -rf .git
git init
```
